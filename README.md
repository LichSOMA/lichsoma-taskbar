# LichSOMA's Taskbar

> **이 모듈은 생성형 AI의 도움으로 제작되었습니다.**

Foundry VTT용 작업표시줄 모듈입니다. Windows 스타일의 작업표시줄을 통해 게임 세션을 더욱 편리하게 관리할 수 있습니다.

## 핵심 기능

### 1. 작업표시줄 커스터마이징
- **색상 및 투명도 설정**: 작업표시줄의 배경 색상과 투명도를 자유롭게 조정
- **높이 조절**: 작업표시줄의 높이를 사용자 환경에 맞게 설정
- **실시간 통계**: FPS 및 지연 시간 모니터링

### 2. 퀘스트 관리 시스템
- 퀘스트 추가 및 수정
- 다중 달성 목표 설정 및 관리
- 퀘스트 상태 관리 (진행 중/성공/실패)
- 목표별 완료 체크 기능

### 3. 사용자 관리
- 접속 중인 사용자 목록 표시
- 오프라인 사용자 표시/숨김 토글
- 사용자 아바타 보기
- 할당된 캐릭터 시트 빠른 열기
- 장면으로 플레이어 불러오기
- **GM 전용 기능**:
  - 플레이어 호출 (사운드 알림)
  - 플레이어 강제 퇴장
  - 플레이어 차단/차단 해제

### 4. GM/플레이어 스크린
- **다중 탭 시스템**: 여러 탭을 생성하여 정보 분류
- **저널 페이지 추가**: 드래그 앤 드롭으로 저널 페이지를 스크린에 추가
- **빠른 편집 모드**: 스크린에서 바로 내용 수정
- **메모 기능**: 각 탭별 메모 작성
- **커스터마이징**:
  - 탭 이름 및 색상 설정
  - 스크린 크기 조절 (가로/세로 칸 수)
  - 격자 표시/숨김
- **아이템 관리**: 액터, 장면, 아이템, 저널, 매크로를 스크린에 추가하여 빠른 접근

### 5. 시스템 메뉴
빠른 접근 버튼을 통한 주요 기능 실행:
- 게임 종료 / 로그아웃
- 페이지 새로고침
- 매크로 관리
- 모듈 설정
- 유저 관리
- 파일 탐색기
- 즐겨찾기

### 6. 기타 편의 기능
- **퀵슬롯 토글**: 필요에 따라 퀵슬롯 숨김/표시
- **바탕화면 보기**: 모든 창을 최소화하여 깔끔한 화면 유지
- **태그 관리**: 항목별 태그 시스템으로 체계적인 관리
- **파일 탐색기 미리보기**: 파일을 선택하기 전에 미리보기

## 설치 방법

1. 압축 파일을 다운로드합니다
2. Foundry VTT의 `Data/modules` 폴더에 압축을 풉니다
3. Foundry VTT를 재시작하고 월드에서 모듈을 활성화합니다

## 기본 사용법

### 작업표시줄 설정하기
1. 작업표시줄의 **설정** 버튼 클릭
2. 색상, 투명도, 높이를 원하는 대로 조정
3. 저장 후 자동으로 페이지 새로고침

### 퀘스트 추가하기
1. 작업표시줄의 **퀘스트** 버튼 클릭
2. **퀘스트 추가** 버튼 클릭
3. 퀘스트 제목과 달성 목표 입력
4. 목표 완료 시 체크박스 클릭

### 스크린 활용하기
1. **GM 스크린** 또는 **플레이어 스크린** 버튼 클릭
2. 사이드바에서 항목을 드래그하여 스크린에 드롭
3. 탭을 추가하여 카테고리별로 정보 정리
4. 우클릭 메뉴로 빠른 편집 또는 시트 열기

### 사용자 호출하기 (GM)
1. **사용자** 버튼 클릭
2. 호출할 사용자 우클릭
3. 호출 메뉴 선택
4. 해당 사용자에게 사운드와 함께 알림 전송
