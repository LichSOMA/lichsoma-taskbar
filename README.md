# LichSOMA's Taskbar

> **이 모듈은 생성형 AI의 도움으로 제작되었습니다.**

Foundry VTT용 작업표시줄 모듈입니다. Windows 스타일의 작업표시줄을 통해 열린 창을 모아 보고, 세션 중 자주 쓰는 기능에 빠르게 접근할 수 있습니다.

## 핵심 기능

- **작업표시줄 커스터마이징**: 배경 **색상**과 **투명도** 조절, FPS·지연 시간 표시
- **퀘스트 리스트**: 퀘스트·달성 목표 추가·수정, 진행/성공/실패 상태 관리
- **사용자 목록**: 접속 사용자 표시, 오프라인 표시 토글, 아바타 보기, 할당 캐릭터 시트 열기
- **호출**: 대상 사용자에게 알림·사운드 전송 (GM은 항상 가능, 플레이어는 모듈 설정에 따라 가능)
- **GM 전용(우클릭 메뉴)**: 장면으로 불러오기, 플레이어 강제 퇴장, 차단/차단 해제
- **GM/플레이어 스크린**: 다중 탭, 저널·액터 등 드래그 앤 드롭, 메모·격자·화면 크기 조절
- **시스템 메뉴**: 게임 종료·로그아웃, 새로고침, 매크로·모듈·유저 관리, 파일 탐색, 즐겨찾기
- **기타**: 퀵슬롯 표시 토글, 바탕화면 보기(창 최소화), 태그 관리, 파일 선택 미리보기

## 설치 방법

이 모듈은 공식 릴리즈를 통해 배포되지 않습니다. GitHub 저장소의 manifest URL로 직접 설치해 주세요.

1. Foundry VTT 셋업 화면에서 **모듈 관리 > 모듈 설치 > Manifest URL** 입력란에 저장소의 `module.json` 직접 링크를 붙여넣습니다.
2. 설치 후 월드에서 **LichSOMA's Taskbar** 모듈을 활성화합니다.

## 기본 사용법

### 작업표시줄 설정

1. 작업표시줄의 **설정** 버튼을 클릭합니다.
2. **색상**과 **투명도**를 조절한 뒤 **저장**합니다.
3. 적용을 위해 페이지가 자동으로 새로고침됩니다.

### 퀘스트 추가

1. 작업표시줄의 **퀘스트** 버튼을 클릭합니다.
2. **퀘스트 추가**로 제목과 달성 목표를 입력합니다.
3. 목표 완료 시 체크박스로 상태를 갱신합니다.

### 스크린 활용

1. **스크린** 버튼으로 GM/플레이어 스크린을 엽니다.
2. 사이드바에서 항목을 드래그하여 스크린에 놓습니다.
3. 탭을 추가·이름·색으로 정리하고, 필요 시 우클릭으로 편집합니다.

### 사용자 호출

1. **사용자** 버튼으로 목록을 엽니다.
2. 호출할 사용자 항목을 **우클릭**합니다.
3. **호출**을 선택하면 대상에게 알림과 사운드가 전달됩니다.
